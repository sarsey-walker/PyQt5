# PyQt5

### Expectativa
Tentei criar uma aplicação em pyqt5 q faz uso da api 'Via Cep' que pega o cep e guarda informações sobre esse cep(nome cidade, bairro, entre outros dispoiveis) se o cep for válido. e persiste essas informaçoes em um banco de dados sqlite3. 
A ideia é ciar uma relacionameno entre a auma tabela q contem as cidades pelo cep e outra com casas disponiveis para alugar na localidade

### Realidade
Consegui a integração com a api(essa foi a minha primeira vez usando uma API yai =^.^= ) e exibe ele em uma tabela. Também é possível graver esses dados numa tabela, (e por emquanto só tem uma e não há botões para tal função) e tbm recupera os dados dessa tabela.
